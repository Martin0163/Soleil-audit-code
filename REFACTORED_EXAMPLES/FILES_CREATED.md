## Installation

1. **Upload to Shopify**:
   - Upload sections to `/sections/` folder
   - Upload snippets to `/snippets/` folder
   - Upload templates to `/templates/` folder

2. **Configure in Theme Editor**:
   - Go to Shopify Admin → Online Store → Themes → Customize
   - Add sections to pages
   - Configure settings

3. **Test Thoroughly**:
   - Check all pages
   - Test on mobile devices
   - Verify cart functionality
   - Test forms and interactions

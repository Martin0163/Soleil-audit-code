## 🚀 DEPLOYMENT INSTRUCTIONS

### Option 1: Manual Upload via Shopify Admin

1. **Backup Current Theme**
   - Go to Shopify Admin → Online Store → Themes
   - Click "Actions" → "Duplicate" on your current theme

2. **Download Refactored Files**
   - Download all files from `/vercel/sandbox/REFACTORED_EXAMPLES/`

3. **Upload Files**
   - Go to Shopify Admin → Online Store → Themes
   - Click "Actions" → "Edit code"
   - Upload each file to its corresponding directory:
     - `theme.css.liquid`, `theme.js`, `ajax-cart.js` → Assets
     - `layout_theme.liquid` → Layout (rename to `theme.liquid`)
     - Section files → Sections
     - Snippet files → Snippets
     - Template files → Templates

4. **Customize**
   - Click "Customize" to use the Theme Editor
   - Configure sections, colors, and content

### Option 2: Using Shopify Theme Kit

```bash
# Install Theme Kit (if not installed)
brew tap shopify/shopify
brew install themekit

# Navigate to your theme directory
cd Soleil.shopify

# Copy refactored files
cp -r /vercel/sandbox/REFACTORED_EXAMPLES/* .

# Rename layout file
mv layout_theme.liquid layout/theme.liquid

# Deploy to Shopify
theme deploy

# Or watch for changes
theme watch
```

### Option 3: Using Shopify CLI

```bash
# Install Shopify CLI (if not installed)
npm install -g @shopify/cli @shopify/theme

# Navigate to your theme directory
cd Soleil.shopify

# Copy refactored files
cp -r /vercel/sandbox/REFACTORED_EXAMPLES/* .

# Rename layout file
mv layout_theme.liquid layout/theme.liquid

# Push to Shopify
shopify theme push

# Or serve locally
shopify theme dev
```

## ⚙️ CUSTOMIZATION GUIDE

### Changing Colors

Edit `theme.css.liquid` (lines 10-20):
```css
:root {
  --color-primary: #FFD700;    /* Change yellow accent */
  --color-black: #000000;      /* Change black */
  --color-white: #FFFFFF;      /* Change white */
}
```

### Changing Fonts

Edit `layout/theme.liquid` (line 45):
```html
<link href="https://fonts.googleapis.com/css2?family=YourFont:wght@400;700&display=swap" rel="stylesheet">
```

Then update `theme.css.liquid`:
```css
--font-heading: 'YourFont', serif;
--font-body: 'YourFont', sans-serif;
```

### Adding Your Logo

1. Go to Shopify Admin → Online Store → Themes → Customize
2. Click on "Header" section
3. Upload your logo image
4. Adjust logo size

### Configuring Navigation

1. Go to Shopify Admin → Online Store → Navigation
2. Edit "Main menu"
3. Add/remove/reorder menu items
4. Save changes

### Setting Up Homepage

1. Go to Shopify Admin → Online Store → Themes → Customize
2. Click "Add section"
3. Choose from available sections:
   - Hero
   - New Arrivals
   - Products
   - Brands
   - Promotions
   - About Us
   - Dreams
4. Configure each section's settings
5. Save and publish

## 🚀 Deployment Quick Start

### Step 1: Backup Current Theme
```
1. Go to Online Store → Themes
2. Click Actions → Duplicate on current theme
3. Rename backup with date
```

### Step 2: Upload New Theme
```
1. Compress "Codigo soleil" folder to ZIP
2. Go to Online Store → Themes
3. Click Add theme → Upload ZIP file
4. Wait for upload to complete
```

### Step 3: Configure Theme
```
1. Click Customize on new theme
2. Follow SETUP_GUIDE.md
3. Configure colors, fonts, logo
4. Add SEO schema snippets
5. Test all functionality
```

### Step 4: Test in Preview
```
1. Test all pages
2. Test cart and checkout
3. Test on mobile devices
4. Verify accessibility
5. Check performance
```

### Step 5: Publish Theme
```
1. Click Actions → Publish
2. Monitor for issues
3. Check analytics
4. Collect user feedback
```

**Detailed instructions:** See SETUP_GUIDE.md
